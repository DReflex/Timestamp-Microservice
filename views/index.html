freeCodeCamp Timestamp Microservice

API Basejump: Timestamp microservice

What is this app for:

1) You can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: January 1, 2017)
2) If it does, it returns both the Unix timestamp and the natural language form of that date.
3) If it does not contain a date or Unix timestamp, it returns null for those properties.

Example usage:

https://reflex-timestamp.gomix.me/January 1 2017
https://reflex-timestamp.gomix.me/1483228800

Example output:

{ "unix": 1483228800, "natural": "January 1, 2017" }
